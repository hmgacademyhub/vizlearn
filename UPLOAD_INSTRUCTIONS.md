# Upload Instructions — Enterprise Folder

This `enterprise/` folder is the deployment-ready package.

## What to upload

When creating a GitHub repository, upload the **contents of this folder**, not the parent folder itself.

Your repository root should directly contain:

```text
index.html
pages/
assets/
docs/
templates/
sw.js
manifest.webmanifest
_headers
_redirects
SECURITY.md
README.md
LICENSE
.nojekyll
.github/
```

## Do not upload only selected files

The platform depends on linked HTML, CSS, JavaScript, manifest, service worker, workflow, docs, and template files. Upload everything in this folder.
