# VizLearn Curriculum Guide — Beginner to Expert Data Visualization Pathway

## Purpose

VizLearn now includes a full data science visualization curriculum designed by taking the role of a data science tutor and curriculum developer. The goal is to move a beginner data scientist from basic chart understanding to expert-level visual analysis, storytelling, model-evaluation visuals, portfolio building, and enterprise reporting.

## New curriculum page

```text
pages/curriculum.html
```

## Curriculum levels

The curriculum is progressive and divided into four levels.

### 1. Beginner

Focus:

- Data science thinking
- Python essentials
- Matplotlib basics
- Seaborn basics
- First EDA workflow
- Chart titles and insight writing

Outcome:

A beginner can read visualization code, create simple charts, inspect a dataset, and explain one insight clearly.

### 2. Intermediate

Focus:

- Distribution analysis
- Relationship analysis
- Categorical comparison
- Time series visualization
- Plotly and interactivity
- Reproducible reports

Outcome:

A learner can choose charts based on data type and analysis purpose, create multi-chart EDA reports, and explain patterns responsibly.

### 3. Advanced

Focus:

- Uncertainty and error bars
- Model evaluation visuals
- Explainability and feature storytelling
- Executive data storytelling
- Dashboard design without paid BI tools
- End-to-end capstone projects

Outcome:

A learner can create professional visual reports, support model interpretation, communicate with stakeholders, and produce portfolio-ready projects.

### 4. Expert

Focus:

- Enterprise data governance
- Cohort analytics
- Visualization design systems
- Teaching visualization

Outcome:

A learner can support institutional training, create reusable visual standards, manage cohort reporting, and teach visualization effectively.

## Tools included on the curriculum page

### 1. Filterable curriculum map

Learners can filter modules by:

- Level
- Track
- Search keyword

### 2. Local progress tracking

Learners can mark modules complete. Progress is saved locally using browser `localStorage`.

### 3. Syllabus export

The full curriculum can be exported as Markdown.

### 4. Curriculum CSV export

The curriculum can be exported as CSV for school planning, Google Sheets, Excel, or enterprise records.

### 5. Study plan generator

The study plan generator creates a weekly plan based on:

- Available hours per week
- Target depth
- Delivery format
- Assessment style

## Why this matters

The curriculum makes the platform beginner-friendly and enterprise-ready. It does not merely provide code snippets. It provides a structured learning journey with outcomes, activities, assessment, and projects.

## No AI API

The curriculum uses static content and JavaScript only. It does not call any AI API.
