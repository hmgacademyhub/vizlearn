## 4. HMG Concepts brand architecture

HMG Concepts is structured around education and technology:

1. **HMG Academy** — virtual learning institution for Nigerian students, covering primary, secondary, WAEC, NECO, GCE, JAMB/UTME, Post-UTME, and other exam preparation.
2. **HMG Technologies** — CBT systems, ML-powered student tools, dashboards, data analysis, school technology setup, EdTech partnerships.
3. **HMG Media** — YouTube, tutorials, social content, public teaching, community engagement.

This architecture should influence VizLearn: it should feel like a teacher-built product, not just a code demo.

## 5. Values to preserve in VizLearn

The platform should continue to reflect these values:

- **Free access:** no paid AI API, no account wall, no server cost.
- **Deliberate learning:** structured progression, not random snippets.
- **Authentic teaching:** every feature should explain why it matters.
- **Practicality:** examples should connect to real data work, especially education and Nigerian context.
- **Transparency:** rule-based helpers should explain their logic; no hidden black box.
- **Mobile-friendly building:** the system should remain maintainable from a browser/tablet.

## 6. Recommended positioning statement for VizLearn

> VizLearn is a free, browser-based Python data visualization learning platform built by Adewale Samson Adeagbo — a Nigerian STEM educator and data scientist with 15+ years of classroom experience. It helps learners move from copying chart code to understanding, modifying, debugging, comparing, and explaining visualizations confidently, using only free static-web tools.

## 7. Enhancement direction based on persona

The best enhancements are not flashy paid AI features. The correct direction is:

1. Add a **simple-to-complex roadmap** because you are a teacher.
2. Add **dataset practice** because you are a data scientist.
3. Add **project/capstone work** because your portfolio is project-driven.
4. Add **storytelling and chart-choice guidance** because you care about understanding.
5. Add **offline/PWA support and backup/export** because your brand values access and zero-cost deployment.
