# VizLearn Enterprise Training Pack Guide

## Purpose

The Training Pack Generator helps a facilitator, school, NGO, academy, or corporate training unit create programme documents without paying for a learning management system.

## Page

```text
pages/training.html
```

## What it generates

The page generates:

1. Training proposal
2. Programme rationale
3. Learning outcomes
4. Weekly scheme of work
5. Assessment model
6. Required free tools
7. Facilitator checklist
8. Data governance note

## CSV templates

The page also downloads blank CSV templates for:

1. Attendance register
2. Completion register
3. Assessment scores

These files can be opened in:

- Microsoft Excel
- Google Sheets
- LibreOffice Calc
- Any CSV-compatible system

## Recommended use

### Before training

1. Generate the training pack.
2. Share the Markdown plan with facilitators.
3. Download attendance and assessment templates.
4. Set up a cohort in the Enterprise Console.

### During training

1. Record attendance weekly.
2. Assign modules from the Curriculum page.
3. Ask learners to export Progress backups.
4. Import backups into Enterprise Console.
5. Track issues in the Issue Register.

### After training

1. Export completion register.
2. Export cohort analytics.
3. Export audit log.
4. Generate certificates where appropriate.

## No paid tools required

The Training Pack Generator uses:

- HTML
- JavaScript
- Browser download APIs
- Markdown
- CSV

No AI API, database, backend, or paid LMS is required.
