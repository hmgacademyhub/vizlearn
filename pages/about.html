HOW VIZLEARN WORKS

Platform Technology

🐍
Pyodide
Python 3.11 compiled to WebAssembly. Runs matplotlib, numpy, seaborn, pandas, plotly entirely in-browser. Zero server, zero cost.
📄
Pure HTML/CSS/JS
No frameworks, no npm, no build step. Static files only — built on an Android tablet using a browser-based workflow.
🐙
GitHub Pages
Free static hosting. GitHub Actions auto-deploys every push to main in ~60 seconds. No server to manage.
💾
localStorage
Progress, quiz scores, streaks, and activity log saved locally. No account needed. Data never leaves your device.
