Guided Learning Roadmap

A deliberate path for data scientists who want to move from syntax copying to independent visual reasoning. The roadmap is fully offline/localStorage based: no account, no paid API, no AI API.

0 / 0 milestones complete

How to use this page: Start from Phase 1, complete the action, tick the box, then move forward. The sequence intentionally moves from chart recognition → syntax → parameter control → comparison → debugging → storytelling → project work.

Recommended weekly structure

Free tools only

Day 1–2: Learn

Read 2–3 lessons and use the Comparator to see the same idea across libraries.

Day 3–4: Practice

Use the Playground, Parameter Explorer, and Dataset Lab. Change one parameter at a time.

Day 5–7: Explain

Take quizzes, fix errors, and build one Story Builder brief or mini-project report.
