Learning Tools

A set of free, local-first study tools that make VizLearn feel like a complete learning system: global search, personal notes, flashcard review, study checklist, and printable certificate. Everything is stored in your browser unless you export it.

No AI API, no database, no login: These tools use static JavaScript, localStorage, and browser print/export features.

Global Platform Search

Static index

Search lessons, features, tools, and references from one place. This is a fast static index, not a server search.

Personal Notes & Bookmarks

localStorage

Save learning notes, useful links, questions, and project reflections. Notes remain private in your browser until you export them.

Flashcard Review

Spaced practice

Review core visualization ideas. Mark each card as Hard, Good, or Easy. The next review date is saved locally.

Visualization Quality Checklist

Before submission

Use this before submitting a chart, project, YouTube tutorial, school report, or portfolio notebook.

Printable Completion Certificate

Print / PDF

Generate a simple certificate for self-paced completion, classroom workshops, or HMG Academy sessions. It is not a formal accreditation system; it is a printable learning record.
